# Autobot Content Factory (Контент-Завод)

Система автоматизации создания и дистрибуции контента для TikTok, YouTube Shorts и Instagram Reels.

## Структура проекта

- `core/`: Базовые классы ботов и общие утилиты.
- `bots/`:
    - `script_bot/`: Генерация идей, сценариев и промптов (OpenAI/Gemini).
    - `visual_bot/`: Интеграция с генераторами изображений и видео.
    - `editor_bot/`: Обработка видео, монтаж, субтитры (MoviePy).
    - `audio_bot/`: Озвучка (TTS) и создание 8D звука.
    - `uploader_bot/`: Автоматизация загрузки контента.
- `data/`: Конфигурации, логи и временные файлы.
- `assets/`: Шрифты, шаблоны, оверлеи.

## Стек
- Python 3.10+
- OpenAI API / Google Gemini API
- MoviePy (видеомонтаж)
- Pydub (аудиообработка)
